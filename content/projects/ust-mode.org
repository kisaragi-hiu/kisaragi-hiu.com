#+title: ust-mode
#+tags[]: utau emacs
#+github: https://github.com/kisaragi-hiu/ust-mode

An Emacs major mode for editing UTAU projects (UST files) as text.

Only provides syntax highlighting (based on =conf-mode=) and a command to normalize paths like OutFile and CacheDir. Useful for searching for a particular flag and replacing all occurances of it, for example.
